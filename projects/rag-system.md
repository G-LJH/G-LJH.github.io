# RAG 知识库检索系统

> *基于 LangChain + 向量数据库的检索增强生成方案*

---

## 项目简介

构建了一套完整的检索增强生成（RAG）系统，实现私有文档的智能问答与精准知识检索。系统支持多种文档格式，通过向量数据库实现语义检索，结合大语言模型生成高质量回答。

---

## 核心技术栈

| 技术领域 | 具体实现 |
|---------|---------|
| 框架 | LangChain 链式调用 |
| 向量数据库 | ChromaDB / Milvus |
| Embedding 模型 | BGE / M3E |
| 大语言模型 | GPT / 通义千问 / 本地模型 |
| 文档处理 | Unstructured IO |

---

## 核心功能

- 📄 **多格式文档支持**: PDF、Word、Markdown、纯文本等
- 🔍 **语义检索**: 基于向量相似度的语义匹配
- 🧩 **分块策略优化**: 自适应上下文窗口切片
- 💬 **多轮对话**: 支持上下文记忆的连续问答
