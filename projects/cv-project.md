# 计算机视觉目标检测项目

> *基于 PyTorch 的深度学习目标检测系统*

---

## 项目简介

基于 PyTorch 深度学习框架，使用 YOLO 系列算法实现的实时目标检测系统。支持图片、视频流、摄像头等多种输入源，具备高精度、低延迟的检测性能。

---

## 核心技术栈

| 技术领域 | 具体实现 |
|---------|---------|
| 深度学习框架 | PyTorch |
| 检测算法 | YOLOv5 / YOLOv8 |
| 图像处理 | OpenCV |
| 数据增强 | Albumentations |
| 部署部署 | ONNX / TensorRT |

---

## 核心功能

- 🎯 **实时检测**: 摄像头 / 视频流实时目标检测
- 📊 **多类别识别**: 支持 80+ 常见物体类别
- 📈 **模型训练**: 自定义数据集训练与调优
- 🚀 **模型加速**: TensorRT 量化与部署优化
