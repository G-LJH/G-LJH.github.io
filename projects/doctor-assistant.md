# 🚀 医生智能助手：基于 FastGPT 的 RAG 工作流

> **项目定位**：针对医疗场景定制的 AI 智能体（AI Agent），通过企业微信 API 实现 7×24 小时全自动化患者咨询服务。

---

## 🏗️ 项目简介

为多位不同专科医生定制开发的 AI 智能助手，通过 FastGPT 工作流实现 7×24 小时自动回复患者咨询。系统能够处理医学问答、挂号指引、购药引导等常见需求，并通过 API 接口集成到企业微信平台，实现自动化患者服务。

### 核心技术栈

- **工作流引擎**：FastGPT 可视化编排

- **对话理解**：意图识别 + 多轮对话管理

- **系统集成**：企业微信 API 对接

## 项目亮点

### 🎯 多专科定制化方案

为每位医生单独定制，确保专业性：
- **知识库独立**：不同专科使用专属医学知识体系
- **工作流定制**：针对性的意图识别与回答策略
- **灵活适配**：深度适配医生个人工作习惯

### 🔗 企业微信深度集成

打通医患沟通渠道：
- 患者通过企业微信直接发起咨询
- AI 自动回复 80% 以上常见问题
- 显著减轻医生日常接待工作负担

### 🧠 完整 RAG 技术体系

从数据到应用的全流程落地：
- 专业医学文档清洗与标准化
- 分层级知识库向量化存储
- 检索增强生成，回答有据可查
